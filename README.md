# Hotel Booking API (ASP.NET Core)

A RESTful API for a hotel booking system built with ASP.NET Core and C#. The project follows clean architecture principles and demonstrates secure, scalable backend development.

---

## Features

- User authentication with JWT and ASP.NET Core Identity  
- Role-based authorization (Admin / User)  
- Manage hotels, rooms, and bookings  
- Prevent overlapping reservations  
- Repository Pattern with DTOs  
- API documentation using Swagger  
- Logging with Serilog  
- Caching and API versioning  
- Clean and layered architecture  

---

## Tech Stack

- ASP.NET Core Web API  
- C#  
- Entity Framework Core  
- SQL Server  
- JWT Authentication  
- AutoMapper  
- Serilog  
- Swagger
